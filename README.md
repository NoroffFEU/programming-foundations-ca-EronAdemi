# Programming Foundations CA
Volvo Vehicle

This object has the following properties;
- Volvo (String), the manufacturer of the vehicle
- S90 (String), the model of the vehicle
- 2020 (number), the manufacturing year of the vehicle
- black (string), the color of the vehicle
- Sunroof (boolean), the sunroof of the vehicle
- features (array), the different features of the vehicle, such as; leather seats, navigation system, park assist.
- honk (function), the horn of the vehicle. If you use this function the vehicle will honk.
